# 🎓 CampusFlow 青春版设计文档

## 设计概述

为 CampusFlow 智慧校园系统打造充满**青春活力**的用户界面，采用鲜艳色彩、圆润字体和卡片式布局，营造友好、活泼的校园氛围。

## 视觉系统

### 色彩方案
```
主色调:    #4F46E5 (靛蓝)      - 按钮、强调元素
次色调:    #818CF8 (浅靛蓝)    - 背景、hover状态
强调色:    #F97316 (活力橙)    - CTA、徽章、高亮
背景色:    #EEF2FF (淡蓝紫)    - 页面背景
文字色:    #1E1B4B (深蓝)      - 主文字
辅助文字:  #64748B (灰蓝)      - 次要信息
成功色:    #10B981 (翠绿)      - 成功提示
警告色:    #F59E0B (琥珀)      - 警告信息
卡片背景:  #FFFFFF (纯白)      - 卡片底色
玻璃效果:  rgba(255, 255, 255, 0.8) - 悬浮卡片
```

### 字体系统
- **标题字体**: Fredoka (Google Fonts) - 圆润、友好
- **正文字体**: Nunito (Google Fonts) - 清晰、现代
- **字号层级**:
  - H1: 32px/2rem (页面标题)
  - H2: 24px/1.5rem (区块标题)
  - H3: 18px/1.125rem (卡片标题)
  - Body: 16px/1rem (正文)
  - Small: 14px/0.875rem (辅助信息)

### 间距系统
- 页面内边距: 24px (桌面) / 16px (移动)
- 卡片内边距: 20px
- 元素间距: 16px
- 组件间隙: 12px

### 圆角系统
- 卡片: 16px
- 按钮: 12px
- 输入框: 10px
- 小元素: 8px
- 圆形按钮: 50%

### 阴影系统
```css
/* 卡片阴影 */
card-shadow: 0 4px 6px -1px rgba(79, 70, 229, 0.1), 0 2px 4px -1px rgba(79, 70, 229, 0.06);

/* 悬浮阴影 */
hover-shadow: 0 10px 15px -3px rgba(79, 70, 229, 0.15), 0 4px 6px -2px rgba(79, 70, 229, 0.1);

/* 按钮阴影 */
button-shadow: 0 4px 14px 0 rgba(79, 70, 229, 0.39);
```

## 交互动效

### 1. 微交互
- **按钮 Hover**: scale(1.02) + shadow增强 + 0.2s ease
- **卡片 Hover**: translateY(-2px) + shadow增强
- **输入 Focus**: 边框颜色过渡到主色
- **消息出现**: fadeIn + slideUp，stagger 0.1s

### 2. 加载状态
- **打字指示器**: 三个跳动的圆点，主色
- **骨架屏**: 渐变动画 shimmer effect

### 3. 滚动效果
- **平滑滚动**: scroll-behavior: smooth
- **新消息**: 自动滚动到底部

## Gradio 自定义 CSS

Gradio 支持通过 `css` 参数注入自定义样式：

```python
custom_css = """
/* 字体导入 */
@import url('https://fonts.googleapis.com/css2?family=Fredoka:wght@400;500;600;700&family=Nunito:wght@300;400;500;600;700&display=swap');

/* 全局样式 */
:root {
    --primary: #4F46E5;
    --secondary: #818CF8;
    --accent: #F97316;
    --bg: #EEF2FF;
    --text: #1E1B4B;
}

/* 聊天消息样式 */
.user-message {
    background: linear-gradient(135deg, #4F46E5 0%, #6366F1 100%) !important;
    border-radius: 16px 16px 4px 16px !important;
}

.bot-message {
    background: white !important;
    border: 1px solid #E0E7FF !important;
    border-radius: 16px !important;
    box-shadow: 0 4px 6px -1px rgba(79, 70, 229, 0.1) !important;
}
"""
```

## 实现优先级

### Phase 1: 核心界面 (高优先级)
1. 应用新色彩系统
2. 集成 Fredoka + Nunito 字体
3. 重设计聊天消息样式
4. 美化快捷入口面板

### Phase 2: 交互优化 (中优先级)
1. 添加微交互动画
2. 优化输入体验
3. 响应式布局调整

### Phase 3: 高级特性 (低优先级)
1. 暗色模式支持
2. 自定义主题切换
3. 音效反馈
