## 📚 文档说明

### 教学文件

#### `ragfiles/` 目录
包含用于测试 RAG 功能的真实文档：

- **2025年本科新生报到手册.pdf** - 新生报到详细指南
- **东华大学学生违纪处分规定.pdf** - 学生违纪处理规定
- **东华大学2026年硕士研究生招生简章.docx** - 硕士研究生招生信息
- **院校简介.docx** - 学校基本情况介绍
- **院系设置及重点实验室.xlsx** - 院系和实验室信息

**使用方式**：
```bash
# 使用这些文档构建 RAG 向量库
python utils/build_rag_from_docs.py
```

## 🚀 快速开始

### 1. 查看 RAG 测试文档

```bash
# 查看文档列表
ls docs/教学文件/ragfiles/

# 查看测试报告
cat docs/测试报告/RAG测试报告-真实文档.md
```

### 2. 构建 RAG 向量库

```bash
# 激活虚拟环境
source .venv/bin/activate

# 使用真实文档构建向量库
python utils/build_rag_from_docs.py
```

### 3. 学习知识点

```bash
# 查看 Git 使用指南
cat docs/知识点补充/Git-GitHub使用指南.md

# 查看环境配置指南
cat docs/知识点补充/环境配置指南.md
```
